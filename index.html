Claude Plugin Marketplace

Extend Claude Code with community plugins. Add this marketplace with one command, then install any plugin instantly.

# 1. Add this marketplace to Claude Code
/plugin marketplace add CoderCoco/claude-plugin-marketplace

# 2. Install a plugin
/plugin install hello-world@codercoco-custom-plugin-marketplace
